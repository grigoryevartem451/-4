# математическая модель
A ∈ ℝ^(m×n) - матрица размером m×n над полем действительных чисел

A = [a_ij], где i = 1,...,m; j = 1,...,n

T: ℝ^(m×n) → ℝ^(n×m)

A^T = [a_ji], где (A^T)_ij = a_ji

+: ℝ^(m×n) × ℝ^(m×n) → ℝ^(m×n)

(A + B)_ij = a_ij + b_ij

×: ℝ^(m×n) × ℝ^(n×p) → ℝ^(m×p)

(AB)_ij = Σ_(k=1)^n a_ik × b_kj
